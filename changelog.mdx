## Upgrading Netra Documentation

Follow these steps to upgrade to the latest version. You run the commands in your project root.

<Steps>
  <Step title="Check Current Version" icon="package">
    Verify your installed version:
    
````bash
npm list netra-docs
````
  </Step>
  
  <Step title="Update Package" icon="download">
    <Tabs>
      <Tab title="npm" icon="npm">
````bash
npm install netra-docs@latest
````
      </Tab>
      <Tab title="yarn" icon="yarn">
````bash
yarn add netra-docs@latest
````
      </Tab>
      <Tab title="pnpm" icon="package">
````bash
pnpm add netra-docs@latest
````
      </Tab>
    </Tabs>
  </Step>
  
  <Step title="Rebuild Site" icon="rocket">
    Rebuild and restart your development server:
    
    <CodeGroup tabs="npm,yarn">
````bash
npm run build
npm run dev
````
````bash
yarn build
yarn dev
````
    </CodeGroup>
  </Step>
</Steps>

<Callout kind="danger">
  Backup your `docs/` folder before upgrading, especially with `breaking` releases.
</Callout>

## Contribution Guidelines

You contribute by following these patterns:

| Area          | Guideline                          | Example Component     |
|---------------|------------------------------------|-----------------------|
| New Features  | Add 5-8 diverse components per page | `<Steps>`, `<Tabs>`  |
